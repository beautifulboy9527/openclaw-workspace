## 数据流示意

```
用户请求
    │
    ▼
┌─────────────────┐
│ 系统管理层      │ ← 路由到正确的 Skill
└─────────────────┘
    │
    ▼
┌─────────────────┐
│ 知识管理层      │ ← 检索历史知识/上下文
└─────────────────┘
    │
    ▼
┌─────────────────┐
│ 分析处理层      │ ← 调用合适的分析 Skill
└─────────────────┘
    │
    ▼
┌─────────────────┐
│ 数据获取层      │ ← 获取原始数据
└─────────────────┘
    │
    ▼
┌─────────────────┐
│ 执行操作层      │ ← 执行具体操作（如需要）
└─────────────────┘
    │
    ▼
返回结果
```

## 核心入口职责

| 入口 Skill | 职责 | 下游 Skills |
|-----------|------|-------------|
| openclaw-control-center | 系统状态监控、任务路由 | 所有系统 Skills |
| knowledge-manager | 知识库操作、写作辅助 | obsidian-cli, wechat-* |
| unified-finance-skill | 金融分析统一入口 | 所有金融 Skills |
| feedgrab | 通用内容抓取 | wechat-*, xiaohongshu-* |
| agent-browser | 浏览器自动化 | 所有需要网页交互的任务 |
| bggg-skill-taotie | Skill 优化进化 | 所有待优化的 Skills |
